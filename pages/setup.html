Setup - Full Styles

Includes all Fonts, Logos, and Icons Packs (Font Awesome & Ionicons)

Download the zip file and extract it. Add the wim.css file to your project directory. Make sure wim.css is the last CSS file referenced in your HTML, especially make sure it's referenced after bootstrap. This stylesheet is compressed and includes all fonts, icons, colors, button styles, etc. This file is compressed and not readable. If you want to make changes, create your own custom CSS file and reference it after wim.css, or download the separate, uncompressed stylesheet partials below.\

You can still use Bootstrap's grids and other elements, available for download at getbootstrap.com
JQuery is also recommended for certain functions. Available at code.jquery.com

After adding wim.css to your project, go back to the .zip file and open the /branding/ directory. This contains all WiM Logos. Move favicon.ico and favicon.png into your project and reference it as the Favicon using: 

<!--[if IE]><link rel="shortcut icon" href="path/to/favicon.ico"><![endif]-->
<link rel="icon" href="path/to/favicon.png"> <!-- Everything else -->

Use the other WiM icons/logos as you need throughout your application.
