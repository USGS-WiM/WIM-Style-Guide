Icons

The icon fonts Font Awesome and Ionicons are both already included in wim.css. Learn how to use them below.

Font Awesome

You can find more information about Font Awesome at fontawesome.github.io.
Icons can be used anywhere (paragraphs, buttons, etc). Just use the following code to use an Font Awesome Icon: 

<i class="fa fa-camera-retro"></i>

Replace fa-camera-retro with whatever class matches the icon you want to use.

See all Font Awesome icon classes here

Ionicons

You can find more information about Ionicons at ionicons.com.
Icons can be used anywhere (paragraphs, buttons, etc). Just use the following code to use an Ionicon: 

<i class="ion-arrow-up-a"></i>

Replace ion-arrow-up with whatever class matches the icon you want to use.

See all Ionicon icon classes here
