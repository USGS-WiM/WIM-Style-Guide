Buttons

The button classes below can be used on <a> or <button> elements.

Button Colors

Add these classes to select color

Default Button
Primary Color Button
Success Color Button
Info Color Button
Warning Color Button
Danger Color Button

Examples:

<button class="btn-wim-danger">Danger Button!</button>
<button class="btn-wim-primary">Primary Button!</button>

Button Sizes

Add these classes along with a color class to change the size

Large Button
Default Button
Small Button
Extra Small Button

Examples:

<button class="btn-wim btn-wim-lg">Large Button!</button>
<button class="btn-wim-primary btn-wim-xs">Extra Small Button!</button>

Text & Icon Buttons

For icon buttons, the icon must come before the text within the button element. Look at the example below. Do not use icons only.

Icon Button
Large Icon Button

Examples:

<button class="btn-wim-primary btn-wim-icon"><i class="ion-ionic"></i>Icon Button!</button>
<button class="btn-wim-info btn-wim-lg btn-wim-icon"><i class="ion-chevron-up"></i>Large Icon Button!</button>

Button Groups

Group buttons (up to 5) in a container side-by-side to fill the width by adding class .btn-wim-group-X where X is 1-5.

Examples:

IMPORTANT:Because of CSS issues, you MUST comment out the space between buttons like shown below)

<!-- Three button group -->
<div class="group-wrapper">
    <button class="btn-wim btn-wim-group-3">3</button><!--
    --><button class="btn-wim btn-wim-group-3">3</button><!--
    --><button class="btn-wim btn-wim-group-3">3</button>
</div>

Inputs

For styling input boxes, text areas, and other form elements.

Text Input

Text input boxes are styled automatically.


Text Input
Password Input
Number Input
Textarea

Examples:

<input type="text" placeholder="Text Input">
<input type="password" value="myPassword#!" placeholder="Password Input">
<textarea placeholder="This is a Textarea"></textarea>

Text Input Sizes

These size classes can be applied to <input> elements or <textarea> elements.


Large Input
Default
Small Input

Examples:

<input type="text" class="input-wim-lg" placeholder="Large Input">
<input type="text" class="input-wim-sm" placeholder="Small Input">

Select Input

Select input elements must be wrapped in a container with class .select-wim. See example below.


Large Select
Default Select
Small Select

Examples:

<!-- All Select Elements must be wrapped in a span wrapper with one of these classes, depending on the size you need:
     .select-wim
     .select-wim-lg
     .select-wim-sm
-->

Example (Large Select)
<span class="select-wim-lg">
    <select>
        <option>Option 1</option>
        <option>Option 2</option>
        <option>Option 3</option>
    </select>
</span>

Other Input Elements

Default styles. No classes needed.


Radio Input
Checkbox Input

Examples:

<input type="radio" id="radio1" /><label for="radio1">Radio</label>
<input type="checkbox" id="checkbox1" /><label for="checkbox1">Checkbox</label>
