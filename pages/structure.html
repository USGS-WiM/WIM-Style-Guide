Lists

Pre-styled <ul> and <ol> lists.


Unordered List

Why use an unordered list
  • When order does not matter.
  • To neatly organize related items.
  • To avoid a long, drawn-out paragraph.
  • Especially when order doesn't matter.

Example Code: 

<!-- Unordered List -->
<h6 class="wim-list-header">My Unordered List<h6>
<ul>
    <li>List Item</li>
    <li>List Item</li>
</ul>

Ordered List

How to make an ordered list
  1. Give it a heading like in the example code below.
  2. Start with the basic <ol> structure.
  3. Think of items that need to be displayed in order.
  4. Add them to the list in order.

Example Code: 

<!-- Ordered List -->
<h6 class="wim-list-header">My Ordered List<h6>
<ol>
    <li>List Item</li>
    <li>List Item</li>
</ol>

Alerts

Alerts should be used to notify a user of an action (successfull submission, error, etc). Normal alerts can be inserted anywhere, and fixed alerts will always appear in the top-right area of the screen.


Normal Alerts

Normal alerts will stretch to match the width of the container

Default Alert! Use for basic info!.wim-alert
Primary Alert! Use for basic info!.wim-alert-primary
Success Alert! Use when action succeeds!.wim-alert-success
Info Alert! Use to provide info!.wim-alert-info
Warning Alert! Use to warn user of issue!.wim-alert-warning
Danger Alert! Use to warn user of error!.wim-alert-danger

Example Code: 

<div class="wim-alert-danger">
    <b>Danger Alert!</b> This is a danger alert!
    <i class="ion-close-round close-alert"></i> <!-- Close Button - MUST INCLUDE -->
</div>

Fixed Alerts

Fixed alerts will always appear in the top-right area of the screen. They must be called by a javascript function to display them.

Fixed Alert!Use for basic info!.wim-alert



Fixed alerts can also be any color. Use classes .wim-alert-fixed, .wim-alert-fixed-primary, .wim-alert-fixed-success, .wim-alert-fixed-info, .wim-alert-fixed-warning, and .wim-alert-fixed-danger.

Example Code: 

<div class="wim-alert-fixed-danger">
    <i class="ion-close-round close-alert"></i> <!-- Close Button - MUST INCLUDE -->
    <b>Fixed Danger Alert!</b>
    This is a fixed danger alert!
</div>

Dismissing Alerts

In order to dismiss alerts, you must include the <i class="ion-close-round close-alert"></i> line within the alert in the same position it is at in the examples above. You must also include the following Javascript code in your main Javascript file in your project (JQuery is required for use). 

$(".close-alert").click(function(){
    $(this).parent().slideUp(250);
});
